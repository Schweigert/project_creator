# ProjectCreator

Create your project from a hash!
```ruby
  {'a': {'hehehe.rb': 'puts "hello world!"'}}.create_project
```

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'project_creator'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install project_creator
